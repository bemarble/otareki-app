# API利用ルール

Spotify APIは

1. アーティスト検索  
2. アーティストの代表曲取得  

のみ使用

- /v1/search  
  - type=artist limit=5
- /v1/artists/{id}/top-tracks  
  - market=JP limit=10

代表曲ページでは

- Cloudflare Worker 経由で /v1/artists/{id}/top-tracks を呼び出し
- 返却された上位10曲について
  - 楽曲画像
  - 曲名
  - Spotify再生リンク（open.spotify.com のURL）

のみ表示すること。

代表曲ページは

- タイムライン結果ページからアーティスト名リンクをクリックしたとき
- 「/artist/:id」 のようなルートで表示する。

# URL設計

タイムラインデータはURLに保存する。

形式

JSON ↓ lz-string compressToEncodedURIComponent ↓ URL
