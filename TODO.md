# TODO.md --- オタレキ MVP 実装タスク

## コンセプト

アーティスト、アイドルのオタク向けのオタク遍歴共有サービス

## サービス概要

「オタレキ」は推してきたアーティスト履歴をタイムライン化し、URLで共有できるWebサービス。

ユーザー操作 
1. アーティスト検索 
2. 推し開始年入力 
3. タイムライン作成
4. URL生成 5. SNS共有

------------------------------------------------------------------------

# 技術スタック

Frontend - React - Vite - TypeScript

Backend - Cloudflare Worker

External API - Spotify API

Storage - URL圧縮 (DBなし)

------------------------------------------------------------------------

# システム構成

User ↓ React SPA ↓ /api/search ↓ Cloudflare Worker ↓ Spotify API

------------------------------------------------------------------------

# 制約

最大アーティスト数

10

理由 - URL長さ制御 - UI安定 - SNS共有最適化

------------------------------------------------------------------------

# URL共有設計

URLは以下の形式

/t/`<compressed>`{=html}

圧縮フロー

JSON ↓ lz-string compressToEncodedURIComponent ↓ URL

------------------------------------------------------------------------

# URLデータ構造

\[ \["artist_name","image_hash","start_year","end_year"\]\]

例

\[ \["YOASOBI","ab6761610000e5ebc3c7","2021",""\],
\["Aimer","ab6761610000e5ebc8f2","2023",""\]\]

image_hash から画像URL生成

https://i.scdn.co/image/`<image_hash>`{=html}
